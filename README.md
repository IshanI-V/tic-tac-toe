# tic-tac-toe

 It's the tic-tac-toe game coded in html, css and javascript. It consista of nine boxes arranged in 3*3 grid, where each box can either contain O or X . 
 this game is played by Two player where players take turn in filling the boxes with X's and O's alternatively. If any player manages to fill three X's or O's
 in a vertically, horizontally, or diagonally arranged boxes, then the game is terminated and that player is declared winner. If no box is left empty and no player 
 wins then the game is declared draw.
